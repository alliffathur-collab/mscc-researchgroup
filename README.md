# MSCC Research Group — GitHub Pages one‑pager

This repository hosts a single‑page website designed for **GitHub Pages**.

The page renders the content from the DOCX file in the repo root:

- `Research Group Website 2025.docx`

## Enable GitHub Pages

1. Go to **Settings → Pages**
2. Under **Build and deployment**
   - **Source**: *Deploy from a branch*
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Save. GitHub will show your Pages URL.

## Edit content

- Update the DOCX file in the repo root.
- The site will automatically render the latest DOCX content when the page loads.

## Local preview

Any static server works (VS Code Live Server extension, etc.).

If your environment blocks running terminals, open `index.html` directly in your browser for a quick preview.
